HTML

Essa é a linguagem base para todo projeto Web. O HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores, são cruciais para um bom rankeamento no google.

JAVA

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90. Hoje a companhia que a mantém é a gigantesca Oracle. Nada melhor do que uma tecnologia que se provou no tempo, certo?

JavaScript

Javascript, também conhecida como JS, é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web e desenvolvimento de software. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Angular

Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript desenvolvido e mantido pela Equipe Angular do Google
